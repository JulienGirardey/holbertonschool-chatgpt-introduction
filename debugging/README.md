# Debugging

Ce dossier contient des exercices de débogage et de résolution de problèmes, réalisés avec l'aide de ChatGPT.

## Objectif

- Corriger des scripts Python ou HTML
- Comprendre les erreurs courantes et leur résolution
- Utiliser l'IA pour améliorer le code

## Fichiers

- `change_background.html` : Manipulation du DOM en HTML/JS
- `checkbook.py`, `factorial.py`, `factorial_recursive.py`, `mines.py`, `print_arguments.py`, `tic.py` : Scripts Python à corriger ou améliorer

## Utilisation

Lire les instructions dans chaque fichier pour exécuter ou corriger les exercices.
