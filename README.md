# Holberton School - ChatGPT Introduction

Ce dépôt regroupe des exercices d'introduction à l'IA générative et à ChatGPT, réalisés dans le cadre du cursus Holberton School.

## Objectif

Découvrir les concepts de base de l'IA générative, du prompt engineering et de l'utilisation de ChatGPT pour résoudre des problèmes variés.

## Structure du projet

- `debugging/` : Exercices pratiques de débogage et de résolution de problèmes avec l'aide de ChatGPT

## Utilisation

Se référer au README de chaque dossier pour les instructions spécifiques à chaque exercice.

## Auteur

Projet réalisé dans le cadre du cursus Holberton School.
